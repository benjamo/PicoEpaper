RasperryPi pico with an EInk Display (GDEY027T91)

SPI Pinout

Raspberry pi pico - gdey027t91

    GP7 (pin 10)  - BUSY
    GP4 (pin  6)  - RST
    GP6 (pin  9)  - DC
    GP1 (pin  2)  - CS
    GP2 (pin  4)  - CLK
    GP3 (pin  5)  - DIN
    GND (pin 38)  - GND
    3V3 (pin 36)  - VCC
